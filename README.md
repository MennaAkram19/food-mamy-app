## 🍔 Food Mamy App

A modern restaurant ordering web application built using **Streamlit** and designed with **Object-Oriented Programming (OOP)** principles.

## 🚀 Features

* Browse menu by category (Main dishes, Sandwiches, Drinks, Desserts, Breakfast)
* Add and remove items from cart
* Apply promo codes (WELCOME10, BIG500)
* Automatic calculation of subtotal, tax, and delivery
* Multi-step checkout system (Cart → Delivery → Confirm)
* Order history tracking
* Interactive and modern UI with animations

## 🧠 OOP Concepts Used

This project is built using **Object-Oriented Programming**, including:

* **Classes & Objects** (MenuItem, Order, RestaurantSystem)
* **Encapsulation** (private price attribute with getters/setters)
* **Inheritance** (MainDish, Drink, Dessert, etc. inherit from MenuItem)
* **Polymorphism** (different item types handled بطريقة موحدة)

## 🛠️ Technologies Used

* Python
* Streamlit
* OOP Design
